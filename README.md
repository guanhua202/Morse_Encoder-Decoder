# Морзе кодировщик/декодировщик

Простой скрипт для преобразования текста в азбуку Морзе и обратно.

## Возможности

- Кодирование текста (латиница) в код Морзе
- Декодирование кода Морзе в текст

Скопируйте файл `morse.py` и запустите в любой среде Python 3.x.
