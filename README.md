# Морзе кодировщик/декодировщик (Примечания к выпуску RU 🇷🇺)

Простой скрипт для преобразования текста в азбуку Морзе и обратно.

## Возможности

- Кодирование текста (латиница) в код Морзе
- Декодирование кода Морзе в текст

## Установка

Скопируйте файл `morse.py` и запустите в любой среде Python 3.x.


git clone https://github.com/guanhua202/morse_encoder-decoder.git
cd morse_encoder-decoder

# Morse encoder/decoder (Release Notes EN  🇺🇸)

A simple script for converting text to Morse code and back.

## Features

- Encoding of text (Latin alphabet) into Morse code
- Decoding Morse code into text

## Installation

Copy the file `morse.py ` and run it in any Python 3.x environment.

git clone https://github.com/guanhua202/morse_encoder-decoder.git
cd morse_encoder-decoder
